# QueryQuill

QueryQuill is a **Smart Help Platform** that provides users with detailed information about various services such as Jobs, Farming, Healthcare, Education, and Government Schemes. Users can explore services through the website or interact with a **LLM-powered chatbot** for detailed assistance.

## Features

- Browse services: Jobs, Farming, Healthcare, Education, Government Schemes  
- LLM-based chatbot for interactive user queries  
- Domain-specific AI responses  
- User login and signup  
- Service-specific pages with detailed information  
- Chat history storage (future enhancement)  


## Technology Stack

**Frontend:**  
- React + Vite  
- JavaScript, CSS  
- Axios for API calls  
- React Router DOM  

**Backend:**  
- FastAPI  
- Python 3.10+  
- SQLAlchemy & MySQL  
- OpenAI API for LLM chatbot  
- JWT for authentication  

**Database:**  
- MySQL (or MariaDB)
